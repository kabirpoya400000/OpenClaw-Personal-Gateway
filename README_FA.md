# راهنمای اجرای برنامه در ویندوز (لوکال)

سلام! برای اینکه زحمات شما به نتیجه برسد و بتوانید از این برنامه بدون مشکل در ویندوز استفاده کنید، این مراحل ساده را دنبال کنید:

### ۱. پیش‌نیازها
مطمئن شوید که **Node.js** روی ویندوز شما نصب است. اگر نصب نیست، از سایت [nodejs.org](https://nodejs.org) نسخه LTS را دانلود و نصب کنید.

### ۲. آماده‌سازی پروژه
بعد از دانلود کد:
1. ترمینال (CMD یا PowerShell) را در پوشه پروژه باز کنید.
2. دستور زیر را بزنید تا پیش‌نیازها نصب شوند:
   ```bash
   npm install
   ```

### ۳. تنظیم API Key (بسیار مهم)
برای اینکه برنامه کار کند، باید کلید خود را به آن معرفی کنید:
1. یک فایل جدید به نام `.env` در پوشه اصلی بسازید.
2. این خط را داخل آن قرار دهید:
   ```env
   VITE_GEMINI_API_KEY=کد_ای_پی_آی_شما
   ```

### ۴. اجرا
دستور زیر را در ترمینال بزنید:
```bash
npm run dev
```
حالا آدرس `http://localhost:3000` را در مرورگر باز کنید.

### ۵. تبدیل به اپلیکیشن ویندوز
1. آدرس بالا را در مرورگر **Chrome** یا **Edge** باز کنید.
2. روی سه نقطه بالا سمت راست کلیک کنید.
3. به مسیر **Apps > Install this site as an app** بروید.
4. حالا برنامه شما مثل یک نرم‌افزار ویندوز در استارت‌منو و دسکتاپ ظاهر می‌شود.

### ۶. حل مشکل Limit
اگر با ارور Limit مواجه شدید:
- وارد تنظیمات (Settings) داخل برنامه شوید.
- بخش **AI Model** را پیدا کنید.
- مدل را از Gemini 2.0 به **Gemini 1.5 Flash** تغییر دهید. این مدل محدودیت بسیار کمتری دارد.

موفق باشید!
